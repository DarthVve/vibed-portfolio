## üöÄ Getting Started

To run this project locally:

1.  **Clone the repository**
    ```bash
    git clone https://github.com/DarthVve/vibed-portfolio.git
    cd vibed-portfolio
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run Development Server**
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  **Build for Production**
    ```bash
    npm run build
    ```
